#   Runtime + optional PyTorch (scaling model inference):

#     pip install -e ".[dev,ml]" --extra-index-url https://download.pytorch.org/whl/cpu
#
# The CPU PyTorch index is optional; omit it if you use CUDA wheels from PyPI or another index.
#

#   - uv:   uv sync --all-extras   (uses uv.lock when present)
#   - pip:  use pip-tools or manual constraints; see README.md
